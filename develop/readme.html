Red Dot Forever 1.04

Red Dot Forever is an extremely simple program to record MIDI files. For advanced editing of MIDI files you'll probably need to look elsewhere — but if you just want to RECORD, listen back to what you PLAYed, and finally SAVE your recording to disk, then Red Dot Forever is all you need. :-)

Tested on Windows XP.

Enjoy this freeware!

If you enjoy using Red Dot Forever, please consider making a donation to help us continue to develop and support this fine program. Thank you for your support!

How to use

It couldn't be easier:

  • Press the big red dot (REC) to start recording. When you're done, press STOP.
  • Not sure if you liked the recording? Press PLAY to hear it again.
  • If you're happy, press SAVE to save a MIDI file to your computer.
  • If you didn't like it, press REC to have another go. This will throw away the old recording.

The program doesn't have more features than that. No metronome, no quantizing, no editing, no... whatever. Just RECORD, PLAY and SAVE.

Options

Press OPTIONS to bring up the configuration screen:

  • MIDI Input: The port that you'll be recording from. It should be connected to your piano or keyboard.
  • MIDI Output: For playing back what you recorded. Use your piano or the software synthesizer that's built into Windows.
  • Echo input while recording: When a MIDI cable is plugged into some keyboards, pressing the keys no longer produces sound. If you enable this option, any incoming MIDI events are immediately sent to the chosen MIDI Output, so you can still hear yourself play (there may be a slight delay). Alternatively, your keyboard may have an option to enable sound again (called "Local Control").
  • Ignore MIDI events: Here you can choose which types of events Red Dot Forever should not record. (Note: Channel Mode and SysEx messages are never recorded.)
  • MIDI triggers: You can assign MIDI events to RECORD, STOP and PLAY. This way you can control Red Dot directly and conveniently from your instrument. You can set the RECORD and STOP triggers to the same event if you wish (though this won't work very well with a pitch wheel, for example).

Extra bonus features

  • For your convenience, a timer keeps track of how long you've been recording already. It starts ticking at the first note you play.
  • Recording won't start until the first actual note is played, so there won't be long silences at the start of the MIDI file.

Shortcut keys

  • R: Start recording (also Enter or Space)
  • P: Start playback
  • X: Stop recording/playback
  • S: Save to MIDI file

Problems?

  • If you get an error message that the MIDI Input or MIDI Output port is in use, then close all other programs that use MIDI (such as audio players) and try again.
  • Some MIDI USB devices seem to act up from time to time. If you keep getting an error message, or if nothing seems to be recorded, unplug your MIDI USB device and plug it back in again and then restart Red Dot Forever. That seems to solve it.

Voices/program changes

Red Dot Forever simply records the MIDI events that are sent to it from your digital piano, keyboard, MIDI controller, etc. When you change a voice on your instrument, it sends a MIDI "program change" event to Red Dot.

This means that if you select a sound, for example organ, before you start the recording, Red Dot does not know you used an organ. The necessary Program Change event that selected the organ voice was sent when Red Dot wasn't yet recording anything.

To incorporate these Program Change events in your MIDI file, first press RECORD and then change the sounds. (Also make sure in the Options screen that Program Change events are not being ignored.)

If you recorded a MIDI file without the Program Changes that you need, you may be able to select the proper sounds on your instrument before playing back the MIDI file. You may also use other software to play the MIDI file, that has options for assinging voices to the MIDI channels. Finally, you can insert your own Program Change events afterwards with MIDI editing software.

Tips

To edit your MIDI files, use Anvil Studio (free).

For playing back your MIDI files, I recommend vanBasco's Karaoke Player or MidiNotate Player or SynthFont (all free).

Version history

  • v1.04, 8-sept-2007:
    • You can now use MIDI "triggers" to start/stop recording and playback. To accomodate this, Red Dot has to open MIDI IN and OUT as soon as the program starts. This can lead to conflicts with other MIDI programs running at the same time.
    • Timer is now also used for playback.
  • v1.03, 4-feb-2007:
    • Made STOP a separate button so you can no longer accidentally double-click it and overwrite your fresh recording with a new one.
    • Moved settings to Options dialog. Settings are now remembered.
    • You can now filter out certain types of MIDI events (like aftertouch).
    • The record timer now doesn't begin until the first NOTE ON event.
    • Clicking STOP now immediately aborts playback (it used to wait until the next event).
    • Added EDIT button but it doesn't do anything yet.
    • Code cleanup.
  • v1.02, 3-oct-2006. Fixed a silly bug that screwed up the timing on long recordings.
  • v1.01, 29-aug-2006. Fixed some text. Included source code.
  • v1.0, 28-aug-2006. First version.
